>* 前后端分离的管理后台
>* 本项目使用的是phthon接口，在[app_admin_api](https://github.com/HJaycee/app_admin_api)
>* 图片验证码需要redis服务配合
>* 图床使用腾讯云COS
>* 使用vue+element-ui构建UI
>* 使用vue-router+vueX配置导航索引
>* 能做到页面刷新前进返回数据不丢失

* 本地运行

```
npm run dev
```

* 配置

```
// src/util/qcloud.js 配置腾讯云cos
// src/util/api.js 配置部署到服务器的接口地址
```

* 部署

```
npm run build
// 将dist静态文件部署到服务器
```
